# Flutter 고급 개발 커리큘럼 최종 요약

## 커리큘럼 완료

Flutter 고급 개발을 위한 5단계 커리큘럼을 모두 완료했습니다.

---

## 전체 커리큘럼 요약

### 01단계: 상태 관리 (Provider)

**학습 내용:**
- Provider 패턴 이해
- ChangeNotifier 사용
- Consumer 위젯
- Provider.of 사용
- 실전 예제 (할 일 목록)

**핵심 개념:**
- 상태와 UI 분리
- 여러 위젯에서 상태 공유
- 효율적인 상태 업데이트

**파일 위치:** `lib/ch01_상태관리/`

**필요한 패키지:**
- provider: ^6.1.1

---

### 02단계: 비동기 처리

**학습 내용:**
- async/await 심화
- FutureBuilder: Future 기반 비동기 UI
- StreamBuilder: Stream 기반 비동기 UI
- 로딩 상태 처리
- 에러 처리

**핵심 개념:**
- 비동기 데이터 표시
- 로딩 인디케이터
- 에러 핸들링

**파일 위치:** `lib/ch02_비동기처리/`

---

### 03단계: 로컬 저장소

**학습 내용:**
- SharedPreferences: 간단한 키-값 저장
- File I/O: 파일 읽기/쓰기
- path_provider: 경로 관리
- 데이터 영구 저장

**핵심 개념:**
- 앱 재시작 후 데이터 유지
- 파일 시스템 접근
- 데이터 저장 패턴

**파일 위치:** `lib/ch03_로컬저장소/`

**필요한 패키지:**
- shared_preferences: ^2.2.2
- path_provider: ^2.1.1

---

### 04단계: 이미지 & 미디어

**학습 내용:**
- image_picker 패키지
- 갤러리에서 이미지 선택
- 카메라로 사진 촬영
- 여러 이미지 선택
- 이미지 표시 및 처리

**핵심 개념:**
- 플랫폼별 권한 처리
- 이미지 선택 및 표시
- 미디어 파일 처리

**파일 위치:** `lib/ch04_이미지미디어/`

**필요한 패키지:**
- image_picker: ^1.0.7

**권한 설정 필요:**
- Android: AndroidManifest.xml
- iOS: Info.plist

---

### 05단계: 외부 패키지 사용

**학습 내용:**
- pub.dev 패키지 검색
- pubspec.yaml에 패키지 추가
- http 패키지: 네트워크 요청
- 패키지 버전 관리
- 패키지 사용법 이해

**핵심 개념:**
- 패키지 의존성 관리
- API 호출
- 외부 라이브러리 활용

**파일 위치:** `lib/ch05_외부패키지/`

**필요한 패키지:**
- http: ^1.1.2

---

## 각 단계별 파일 구조

각 단계는 다음 파일들로 구성됩니다:

```
lib/ch0X_단계명/
├── main.dart      # 실행 가능한 Flutter 앱 예제
├── 설명.md        # 상세한 설명과 개념
└── 실행.md        # 실행 방법 가이드
```

---

## 실행 방법

각 단계의 앱을 실행하려면:

```bash
cd ch03_advanced
flutter pub get
flutter run lib/ch0X_단계명/main.dart
```

예시:
```bash
flutter run lib/ch01_상태관리/main.dart
flutter run lib/ch02_비동기처리/main.dart
flutter run lib/ch03_로컬저장소/main.dart
flutter run lib/ch04_이미지미디어/main.dart
flutter run lib/ch05_외부패키지/main.dart
```

## 학습 순서

1. **01단계: 상태 관리** - Provider로 상태 관리 방법 학습
2. **02단계: 비동기 처리** - FutureBuilder, StreamBuilder로 비동기 데이터 처리
3. **03단계: 로컬 저장소** - 데이터 영구 저장 방법 학습
4. **04단계: 이미지 & 미디어** - 이미지 선택 및 처리 방법 학습
5. **05단계: 외부 패키지 사용** - 외부 패키지 활용 방법 학습

- 패키지 추가 및 사용

- JSON 처리

## 사용된 주요 패키지

1. **provider** (^6.1.1)
   - 상태 관리
   - ChangeNotifier 기반

2. **shared_preferences** (^2.2.2)
   - 간단한 키-값 저장
   - 사용자 설정 저장

3. **path_provider** (^2.1.1)
   - 파일 시스템 경로 관리
   - 앱 문서 디렉토리 접근

4. **image_picker** (^1.0.7)
   - 이미지 선택
   - 카메라 촬영

5. **http** (^1.1.2)
   - HTTP 요청
   - API 호출
